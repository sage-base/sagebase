# SKILLを積極的に活用すること

タスクに取り組む際は、`.claude/skills/` 配下に定義されたSKILLをできるだけ活用してください。

## ルール

- タスクの内容が既存のSKILLのアクティベーション条件に該当する場合は、**必ず該当SKILLをアクティベートしてから作業を開始**すること
- 複数のSKILLが該当する場合は、すべてアクティベートすること
- SKILLの内容に従い、チェックリストやガイドラインを遵守すること
- 不明な点がある場合は、まずSKILLの内容を確認してから判断すること

## 利用可能なSKILLの確認方法

- `.claude/skills/` ディレクトリ配下のサブディレクトリがそれぞれ1つのSKILLに対応
- 各SKILLのディレクトリ内にある `.md` ファイルにアクティベーション条件と詳細なガイドラインが記載されている
